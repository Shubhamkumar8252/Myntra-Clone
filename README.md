#Myntra Clone
A simple Myntra homepage clone built using HTML and CSS, focusing on layout, design, and UI structure.

#Features
Responsive header and navigation bar.
Search bar with icons.
Profile, Wishlist, and Bag sections
Banner and category sections.
Clean footer layout.

#Tech Stack
HTML5
CSS3

#Purpose
This project was created for learning and practicing frontend development and understanding real-world website layouts.
